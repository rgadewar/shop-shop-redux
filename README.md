![GitHub license](https://img.shields.io/badge/license-MIT-red)
  
## Description 

Challenge is to refactor the e-commerce platform from Activity 26 Links to an external site. so that it uses Redux Links to an external site.. You won’t need to make sweeping changes to the code, but you will need to read through the Redux documentation on your own to find the information you need. We’ve provided some guidelines to point you in the right direction in the Getting Started section below.
